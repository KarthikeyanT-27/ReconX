##**ReconX   [Cyber Fingerprinting Toolkit]**##

🎯 The Right Tool for Authorized Reconnaissance
This modular, professional toolkit bundles essential scripts for learning and performing authorized cybersecurity fingerprinting and reconnaissance. Stop juggling scattered tools—get reliable, color-coded results from a single command-line interface.

🚀 Core Capabilities:

    🌐 Network : "Live Host Discovery (ARP), OS Fingerprinting (TTL), TCP Port Scanner, Traceroute." -- Ideal for internal network assessment.
    🔎 Reco : "DNS/Reverse DNS, WHOIS Lookup, Comprehensive Subdomain Enumeration.", -- Gathers essential external intelligence.
    ☕ Web : "Web Server Header Analysis, SSL/TLS Certificate Inspection.", -- Pinpoint web server configuration details.

▶️ Quick Demo:

      Running python3 ReconX.py gives you an immediate, structured menu:

🛠️Installation & Setup:
        1. Clone the Repository:
            git clone https://github.com/KarthikeyanT-27/ReconX
            cd ReconX
        2. Install Dependencies:
            pip install -r requirements.txt
